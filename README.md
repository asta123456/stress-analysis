# Sleep Stress Prediction App

## About the Project
This project predicts a person's **stress level** based on their **sleep, mobile usage, and lifestyle habits**.

It uses Machine Learning and is deployed using **Streamlit**.

---

## Features Used
- Age
- Gender
- Occupation
- Daily screen time
- Phone usage before sleep
- Sleep duration
- Sleep quality
- Caffeine intake
- Physical activity
- Notifications received per day
- Mental fatigue score

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn
